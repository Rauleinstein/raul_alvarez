Napakalaki

Napakalaki es una aplicación en Java que pretende emular al juego de cartas Munchkin Cthulhu. Mantiene un sistema clases que controlan todos los parámetros del juego y está diseñado desde una perspectiva de programación dirigida a objetos. El código de la aplicación es libre y está totalmente disponible en GitHub

Desarrollo

En el desarrollo participamos 3 alumnos, donde cada uno se encargaba de una parte del proyecto. El proyecto constaba de partes tanto de diseño de la estructura como de codificación de dicho diseño.
